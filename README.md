# FitTrack

A very simple exercise tracker designed to work without any accounts

## Features

- [x] Add exercises
- [x] Add workouts
- [x] Add sets
- [x] View history
- [x] View trends
- [x] Import/Export data as CSV
- [x] Dark mode
